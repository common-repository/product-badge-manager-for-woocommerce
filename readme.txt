== Description ==

Product Badge Manager For Woocommerce lets you create unlimited product badges and assign them to your WooCommerce product based on category, attributes, terms, popularity, sale, time, stocks, etc, and filter products archive using badges.

== Free Features ==

 - Show Badge On Single Product Page
 - Show Badge On Product Catalog Pages
 - Assign Badge By Product Taxonomies
 - Assign Badge Based On Category
 - Assign Badge Based On Seller
 - Assign Badge Directly From Product
 - Floating Badge Top Of Product Image

== Installation ==

There is no prerequisites for installing **Product Badge Manager For WooCommerce** plugin. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Product Badge Manager For WooCommerce`.
3.  Install and activate the plugin, that's it!

= Manual Installation =
1.  Download [Product Badge Manager For WooCommerce](https://downloads.wordpress.org/plugin/product-badge-manager-for-woocommerce.zip "Product Badge Manager For WooCommerce") plugin.
2.  Extract the `product-badge-manager-for-woocommerce.zip` file. You'll get plugin files inside `product-badge-manager-for-woocommerce` directory.
3.  Upload the plugin files to the `/wp-content/plugins/product-badge-manager-for-woocommerce` directory.
4.  Activate the plugin through the 'Plugins' screen form your WordPress website dashboard.
